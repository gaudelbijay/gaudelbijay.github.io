Projects

Image Quality Assessment

I did this project under the supervision of Prof. Ukash Nakarmi from university of Arkansas. In this Project, we explore several convolutional neural network-based frameworks and different data augmentation technique for medical image quality assessment. We used nearest neighbor loss function as a regularizer which helps to find the best model in less epoch. We used Wavelet transform methods to decompose image into images of same classes and train the model.

Image Reconstruction

I am working on this project under the supervision of Prof. Bikash Nakarmi and Prof. Ukash Nakarmi. We have created image from k-sampled data from radar system using back projection methods. We have further extended our experiment and working on deep learning approaches to improve our model performance.

Nepali Text Generation} {Nepali Text Generation is a deep learning framework (using Pytorch) for suggesting relevant word during writing. We used Recurrent Neural Network with layer GRU or LSTM, and KL divergence as loss function to train our model. Code is avilable here.

Implementation of GraphSAGE framework

GraphSAGE is a bench mark framework that leverages node feature information (e.g., text attributes) to efficiently generate node embeddings for previously unseen data. I implement this framework using TensorFlow 2.x. Code is avialable here.

Remote Sensing on Hydroponics System

This project is the part of complete hydroponics system and was funded by Nepal Government Bagmati Pradesh. I worked mostly on remote sensing part of the project. In this project I developed mobile app to track sensor data from hydroponics system. I built REST API for back-end using Flask framework to stream data continuously and mobile app using flutter framework to visualize sensor data remotely.Sample code of the project is avialable here.
