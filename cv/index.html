Projects

  • Joint-Angle Human Pose Estimation for Casualty Manipulation
    • Built a depth-only model that estimates human joint angles and body-link classes from single frames or short frame windows.
    • Added feasible-state prediction so the model returns physically valid joint-state sets, not only one point estimate.
    • Developed a staged training pipeline for angle prediction, ranking, and uncertainty modeling to improve reliability.
    • Implemented reproducible split tracking and multi-GPU server training workflows using SLURM job scripts.
  • ROS 2 Haptic Teleoperation Data Logger for Manipulation Learning
    • Built a ROS 2 teleoperation package for controlling robot-arm experiments with a Geomagic Touch haptic device.
    • Implemented configurable button mapping, launch workflows, and control support for a custom-designed hand gripper.
    • Created labeled recording tools that store rosbag2 data and metadata by grasp target for supervised training.
    • Added dataset-export scripts that convert recorded sessions into depth samples and CSV labels for model training.
  • Real-Time Human Pose Refinement with EKF and Kinematic Constraints
    • Developed a real-time ROS 2 module that refines noisy 2D human keypoints using EKF filtering and kinematic constraints.
    • Improved pose stability by rejecting outliers and enforcing consistent limb-length and joint-angle relationships.
    • Published refined pose outputs, diagnostics, and optional annotated images for debugging and monitoring.
    • Designed launch and runtime options for simulation and robot use cases, including configurable annotator behavior.
  • ROS 2 + Gazebo Harmonic Casualty Manipulation Simulation Stack
    • Built an end-to-end ROS 2 Jazzy and Gazebo Harmonic simulation stack for collaborative robot and human-interaction research.
    • Simulated a Clearpath Husky with UR manipulators and a compliant human model to support contact-rich manipulation scenarios.
    • Added single-arm and dual-arm operating modes and integrated a custom-designed hand gripper configuration.
    • Documented full setup, dependency installation, and launch workflows so experiments can be reproduced across systems.
  • Event-Camera Reinforcement Learning Navigation on Jackal
    • Built an event-camera navigation framework that learns from asynchronous event streams instead of standard image frames.
    • Implemented compact event representations and contrastive perception features for reinforcement learning policies.
    • Trained and compared MLP, CNN, GRU, and Transformer agents using a Soft Actor-Critic training setup.
    • Demonstrated competitive simulation results and successful transfer to a real Jackal robot with minimal retuning.
  • Diffusion-Based Defense Against Adversarial Perception Attacks
    • Developed a diffusion-based denoising pipeline to reduce the impact of online adversarial image attacks.
    • Integrated an FFT-based detector to flag attack patterns during runtime.
    • Connected the defender to an RL attacker workflow in ROS and AirSim by routing perception through denoised images.
    • Built repeatable training and runtime workflows with Python virtual environments and catkin integration.
